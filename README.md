# 电商用户行为分析

## 项目简介
基于淘宝用户行为数据，分析用户从浏览到购买的转化路径，应用RFM模型进行用户分群。

## 技术栈
- Python 3.8+
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 运行方法
1. 安装依赖：`pip install -r requirements.txt`
2. 打开Jupyter Notebook：`jupyter notebook 电商用户行为分析.ipynb`

## 分析结果
- 整体购买转化率：69.4%
- 识别出高价值用户、潜力用户、一般用户、流失风险用户四类群体
